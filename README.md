# Sewa Management System

A full-stack workflow automation platform built to replace manual record-keeping and reporting processes for sewa operations.

The system manages 900+ real-world records and streamlines operational workflows including member management, search-based autofill, structured document generation, and reporting.

## 🚀 Live Demo

https://sewa-management-system-njna.vercel.app/

---

# ✨ Features

- Role-based authentication (Admin & Demo users)
- Multi-database architecture for demo and production environments
- Advanced people search with debouncing and Elasticsearch-style search experience
- Search-based autofill for rapid data entry
- Dynamic sheet and record management
- Structured nominal roll generation
- PDF export functionality
- Responsive UI optimized for desktop and mobile
- Real-world workflow automation replacing manual Excel/Word processes
- Secure session-based authentication
- Cloud deployment with Vercel

# 📌 Real-World Use Case

This system was built for real operational usage to digitize and automate sewa management workflows previously handled manually through Excel sheets and Word documents.

The application currently manages:
- 1000+ production records
- Structured reporting workflows
- Automated document generation
- Operational data organization

---

# ⚡ Key Engineering Highlights

- Implemented debounced search to minimize unnecessary API requests and improve performance
- Designed scalable multi-database architecture supporting separate demo and production environments
- Built optimized autocomplete search experience for handling large datasets
- Implemented secure session-based authentication using signed cookies
- Developed printable document generation and PDF export workflows
